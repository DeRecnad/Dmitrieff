History-Runes

History-Runes – это образовательный сайт, посвящённый славянским рунам. На сайте представлены изображения славянских рун, их описания и значения. Этот проект создан для того, чтобы помочь пользователям лучше понять и изучить древнюю письменность славянских народов.

BusTimeTable

BusTimeTable – это удобный веб-сайт для поиска маршрутов и расписаний автобусов, электричек и поездов в Пермском крае. В процессе работы над этим проектом я получил ценный опыт взаимодействия с API для интеграции данных о транспорте в реальном времени. Также я научился работать в команде из двух человек, что помогло мне развить навыки коммуникации, совместного планирования и координации задач. Этот проект улучшил моё понимание веб-разработки, а также позволил мне освоить принципы интеграции внешних данных в пользовательские приложения.

Wiki_Thief

Wiki_Thief – это мощный инструмент для автоматизации процесса переноса файлов между различными вики-системами. В ходе работы над этим проектом я получил опыт использования библиотек Selenium и requests для автоматизации веб-задач и взаимодействия с веб-страницами. Я научился эффективно работать с динамическим контентом, разрабатывать парсеры для извлечения текстовых данных и медиафайлов, а также решать задачи по обработке и переносу больших объемов данных. Этот проект помог мне лучше понять процессы автоматизации и управления контентом в разных системах.

Parma_Cafeteria

Parma_Cafeteria – это инновационный кафетерий, созданный по заказу PARMA Technologies Group в рамках проекта в высшем учебном заведении. Этот кафетерий предоставляет уникальные льготы для сотрудников, основанные на их достижениях и грамотах. В процессе работы над проектом я получил опыт работы в команде.Научился работать с базами данных и интеграции их в корпоративные системы, что позволило создать надежную и удобную для пользователей платформу.

HidenSeek

HidenSeek – Инструмент для поиска аккаунтов по User ID в интернете и просмотра метаданных фотографий. Во время разработки получил большой опыт работы с Flask и Django. Так-же получил опыт работы с различными API.
